# 247 Sports (tfs) python client

## Installation

recruitR-py can be installed via pip:

```bash
pip install recruitr
```

or from the repo (which may at times be more up to date):

```bash
git clone https://github.com/saiemgilani/recruitR-py
cd recruitR-py
pip install -e .
```
